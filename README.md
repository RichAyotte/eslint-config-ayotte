# eslint-config-ayotte
Ayotte Software JavaScript Style Guide

*Probably a different coding style than you're used to.*

## Usage
In your `.eslint.js`

```javascript
'use strict'

module.exports = {
	extends: [
		'ayotte'
	]
}
```
